# Proyecto-Web-Ejemplo

Este es un proyecto para afianzar conceptos del curso adicional de git.

Para clonar el repositorio simplemente obtén el enlace ya sea via SSH(nestro caso) o HTML. Una vez en la
terminal UNIX introduce git clone <enlace>.

# Ramas y funcionalidad

En este repositorio de prueba existen 2 ramas, la rama principal y la rama de cambios en el estilo, ambas
tienen conflictos y a través de la gestion en el merge se obitiene el producto deseado.
